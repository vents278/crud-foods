# 🍽️ CRUD Foods

Aplicação web de cadastro e gerenciamento de alimentos, desenvolvida com Spring Boot no back-end e HTML/CSS/JavaScript no front-end. Ideal para quem busca aprender ou aplicar conceitos de CRUD (Create, Read, Update, Delete) em aplicações web.

## 🚀 Funcionalidades

- ✅ Listagem de alimentos
- ➕ Cadastro de novos alimentos
- ✏️ Edição dos alimentos existentes
- ❌ Remoção de alimentos


---

## 🛠️ Tecnologias Utilizadas

- Java 17+
- Spring Boot
- Spring Web
- Spring Data JPA
- MySQL
